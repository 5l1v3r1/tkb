# 高质量技术分享小组 - 每日更新

分享高质量有价值的技术信息, 包括不限于各种技术资讯, 小技巧, 黑科技, 骚操作.

每日在微信群分享至少一条有价值的技术资讯, 每月发布月度 PDF, 每年发布年度 PDF.

# 目录组织

月度或者年度的 PDf 位于 `release` 目录, 欢迎下载阅读.

如果涉及到相关资料, 将会存放于 `datum` 目录, 资料名称的前缀是日期.
